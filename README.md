# React Native Custom Component

## Usage :

- Every component name should be start with <b>Capital latter</b> :

```js
const CustomComponent1 = () => {
  return (
    <View>
      <Text>Your CustomComponent 1</Text>
    </View>
  );
};
```

- You can use this component into your app like this :

```js
<CustomComponent1 />
```
